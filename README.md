# Real-Time Chat Application

## 📌 Overview
This project is a real-time chat application that enables multiple users to communicate instantly without page refresh. It uses WebSocket technology to ensure fast and efficient message delivery between clients and server.

---

## 🚀 Features
- 💬 Real-time messaging between multiple users
- 🌐 Instant message broadcasting
- ⚡ No page refresh required
- 👥 Multi-user support
- 🔄 Live updates using WebSocket
- 🎯 Simple and responsive user interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Socket.io

## ⚙️ Installation

1. Clone or download the repository  
2. Open the project folder in terminal  
3. Install dependencies:


npm install


---

## ▶️ Run the Application


node server.js


Then open in browser:


http://localhost:3000


---

## 🧪 Usage

- Open the application in multiple tabs or browsers  
- Enter messages in one window  
- Messages will appear instantly in all connected users  
- No page refresh required  

---

## 🧪 Testing

- Open 2–3 browser tabs  
- Send messages from one tab  
- Verify real-time updates in other tabs  
- Check console for any errors
